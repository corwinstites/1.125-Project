WELCOME TO SEASOUND


By Corwin Stites and Edvard Rongaln - December 01, 2022

Hi there, and welcome to SeaSound! SeaSound is an interactive tool for predicting sound velocity profiles near the U.S. coast. Make yourself a cup of tea, pull up a chair, relax, and peruse our website! To use SeaSound, click the 'Use the Tool!' button. To learn about the data we used to train our model, click the 'Checkout the Data!' button. To learn about the guys who made this, click the 'Meet the Creators!' button. If you want to learn more about sound velocity profiles and why they matter, click the 'Learn About Ocean Sound!' button. Click the 'SeaSound Home' button to return to this page. Click on our names or pictures to check out our GitHub accounts. If you want to see all the code for the project on GitHub click the button below. Also check out our acknowledgments for all the images, source code, information, and other assistance we got for this project below! Thanks for visiting SeaSound and we hope you enjoy using it!


-Sincerely,

The SeaSound Team
